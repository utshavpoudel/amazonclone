# 🛒 Amazon Clone

A responsive front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. This project recreates the design and layout of Amazon's landing page to practice front-end web development skills.

## 📌 Features

- Responsive navigation bar
- Search bar
- Hero section
- Product category cards
- Product grid layout
- Footer with multiple sections
- Clean and organized code

## 🛠️ Built With

- HTML5
- CSS3

## 🎯 Learning Objectives

This project helped me practice:

- Semantic HTML
- CSS Flexbox
- CSS Grid
- Responsive Web Design
- Layout Design
- UI Development

## 🔮 Future Improvements

- Add JavaScript for interactivity
- Implement a shopping cart
- Add product search functionality
- Improve responsiveness for all screen sizes
- Connect to a backend for dynamic content
